# AI-Based Sentiment Classification for Customer Reviews

## Project Description

This project focuses on analyzing customer reviews and classifying them into sentiment categories using machine learning techniques. The goal is to automatically determine whether a review expresses a positive, negative, or neutral opinion.

## Technologies Used

* Python
* Scikit-learn
* Google Colab

## Machine Learning Models

Several machine learning models were implemented and compared in this project:

* Random Forest
* Multinomial Naive Bayes
* Support Vector Machine (SVM)
* Stacking Ensemble

## Methodology

The dataset of customer reviews was processed and transformed into numerical features suitable for machine learning models. Multiple classification algorithms were trained and evaluated to compare their performance. A stacking ensemble model was then implemented to combine the predictions of multiple models and improve the overall accuracy.

## Results

The stacking ensemble model achieved an accuracy of **0.83**, showing better performance compared to individual models.
